Project X

For my final project this semester, I chose to develop a webpage using HTML, CSS, and JavaScript to deepen my development skills. I created a website showcasing diverse art cultures from different countries and regions worldwide.

My Contributions

I developed a website showcasing different art cultures worldwide. I added a 3D globe on the homepage that spins and moves with the cursor . For development, I used JavaScript, Node.js, Three.js, GSAP, and Vite.

I installed Node.js to manage dependencies and used Three.js to create a 3D sphere, set up a scene, and add a camera for rendering. I applied shaders and textures to make the sphere look like Earth, defining shaders in a GLSL file and importing them into JavaScript. With GSAP, I added mouse movement functionality, making the 3D globe follow the cursor and rotate. I used Vite as a local server to facilitate development and testing.

I learned how to work with Three.js, including setting up a scene and camera, rendering, and modifying values.
